# Portfolio assets

Add only company-approved or redacted public files here.

- Save your clear profile photo as `profile.jpg`.
- Use descriptive filenames for evidence, for example `redacted-dashboard.png`.
- Never commit passwords, tokens, customer information, internal URLs, proprietary source code, or unapproved workplace screenshots.
- Large evidence files are usually better hosted on an approved public service, then linked from `portfolio_content.json`.
